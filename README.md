# KIYAFET SITESI - Frontend Project
Bu proje, modern frontend teknolojileri kullanılarak geliştirilmiş, kullanıcı dostu bir arayüz çalışmasıdır. Profesyonel gelişim sürecimin bir parçası olarak tasarlanmış ve optimize edilmiştir.

# Öne Çıkan Özellikler
* **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumlu.
* **Modern UI/UX:** Kullanıcı deneyimi odaklı, temiz ve sade arayüz.
* **Performans:** Hızlı yükleme süreleri için optimize edilmiş kod yapısı.

# Kullanılan Teknolojiler
* **Framework:** React.js
* **Styling:** CSS3 / Tailwind CSS
* **Package Manager:** npm

# Kurulum ve Çalıştırma
Projeyi yerel bilgisayarınızda çalıştırmak için şu adımları izleyebilirsiniz:
1. Bu repoyu klonlayın:
git clone [https://github.com/kullanici-adin/repository-adin.git](https://github.com/kullanici-adin/repository-adin.git)
2. cd hm-hompage
3. npm install
4. npm run dev
